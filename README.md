# statcrawler
We aggregate the data from all your favorite services and provide a central UI that displays all of them. Status pages we offer are 
- Zoom
- GoToAssist
- Egnyte
- Notion
- AWS
- Mimecast
- Druva
- Adobe
- Slack
- MS365
- Zendesk 
- Jamf
